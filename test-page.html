How to Test

1. Load the Animation Builder extension in Chrome
2. Click the extension icon to open the popup
3. Click "Select Element" to enter selection mode
4. Click on any element on this page to select it
5. Choose an animation type and settings
6. Click "Add to Timeline" to add the animation
7. Use "Play" to see the animation on this page!
